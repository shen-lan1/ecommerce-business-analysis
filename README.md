# Ecommerce Business Analysis Project

## Background
This project analyzes 50,000+ ecommerce orders to identify profitability structure issues and optimization opportunities.

## Key Analysis
- Data cleaning & anomaly detection
- Sales & profit trend analysis
- Category & sub-category structure breakdown
- Discount vs. sales validation
- Regional profitability analysis

## Key Findings
- Tables sub-category shows structural loss (Profit Margin -24%)
- High discounts do not significantly increase sales but reduce profit
- Southeast Asia region has low profitability efficiency

## Recommendations
- Optimize discount strategy for loss-making sub-categories
- Adjust resource allocation toward high-margin products
- Improve product mix in low-profit regions
